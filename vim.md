Shortcuts for vim text editor:

```
i - start 'insert' (edit) mode
ESC - stop 'insert' mode (back to operation)
:w<enter> - save
:q<enter> - quit
:wq<enter> - save & quit
:q!<enter> - quit without saving

Operation:
----
d - delete 
c - change (deletes and start 'insert' mode)
> - indent
y - copy 

u - undo
. - repeat last operation

Subject:
----
w - word
iw - inner word
it - inner tag
i" - inner quotes
ip - inner paragraph

Finding:
-----
F - 
/ - search 



```
